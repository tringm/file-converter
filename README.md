# file-converter
A simple Python implementation for file converter
Available converter:
1. On top of pandas: durable for small file


### Instalation
1. Create virtual environment with [Pipenv](https://pipenv.readthedocs.io/en/latest/) using the Pipfile
1. Run:
    ```python
    python main.py -h
    ```
    for more instruction
    * ***NOTE***:
        * The converter log will be stored in io/out/converter.log
    
2. Test:
    ```python
    python test.py -h

    ```
    * ***NOTE***: 
        * The metrics log file will be stored in test/io/out/metrics.log
